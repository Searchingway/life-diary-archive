# Windows Packaging

This folder keeps the Windows installer source and PyInstaller specs only.

Generated files such as EXE installers, app bundles, build folders, and payload ZIP files should stay out of Git. Use `scripts/build_windows.ps1` to rebuild them locally, then upload the final installer to GitHub Releases.

Expected local payload path during installer packaging:

```text
packaging/windows/payload/life_diary_payload.zip
```

That payload is generated by `scripts/build_windows.ps1` and ignored by Git.
